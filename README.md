# sleep-wakeup

상황
1. 게임 플레이 도중 hold키로 백그라운드 전환 시 게임이 종료되는 현상 발생
2. 게임이 종료되어 다시 켜야하는 UX 문제 발생

과제
1. 게임 실행 후 스크립트를 수동 실행
2. 스크립트에 정한 시간 동안 Sleep Wake up을 반복하여 게임이 종료되는지 확인하여야 합니다.

진행 과정
1. 기기 연결 확인 - adb devices로 감지된 디바이스 확인
2. (y/n) 입력하여 테스트 진행 여부 확인
3. 설정된 시간으로 반복 테스트 진행 확인
4. 디바이스 감지 x, 진행 여부 n 입력, 디바이스 전원 종료, 설정 시간 종료 시 테스트 종료 후 log 파일 추출
5. 저장된 메모장으로 테스트 결과 확인

결과
1. 게임을 실행 후 백그라운드 전환 시 얼마만큼의 시간이 지난 후 크래시 발생하는지 정확한 수치 확인 가능
2. 이슈 리포트 시 간헐적이 아닌 정확한 근거 제시 가능
3. QA 인력이 단순 반복 작업 대신 다른 업무에 집중 가능
